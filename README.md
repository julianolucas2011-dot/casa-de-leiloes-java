## Sistema de Leilão/Atividade 1 UC 11

## Esse projeto é baseado em um sistema informatizado para uma casa de leilões, visando automatizar as rotinas de segurança e qualidade de software, conforme exigido pela empresa.

## Tecnologias utilizadas:

* Linguagem de Programação: Java
* Sistema de Gerenciamento de Banco de Dados: MySQL
* Versionamento: Git e GitHub
